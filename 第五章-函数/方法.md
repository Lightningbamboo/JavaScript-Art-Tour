### 作为成员的函数

我们在上一节中已经了解过，JavaScript 中的函数也是一种值，它们可以被赋给其他标识符，自然，函数也可以成为对象属性的值。当一个函数成为一个对象的成员时，我们可以叫它*成员函数*，更常见一点的叫法是*方法*。

按照我们的理解，对象中的方法看起来应该是这样的：

```javascript
let person = {
    name: "Jason",
    say: () => alert("Hello world");
}
```

那么我们就可以调用这个方法：

```javascript
person.say(); // "Hello world"
```
